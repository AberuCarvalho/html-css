Seletores ID e Class

Os seletores são metodos de estilizar certas partes do HTML de uma forma mais específica, fora dos comandos gerais das CSS. Assim o seletor ID ele é único e só pode ser usado em uma pagina e apenas uma vez no código HTML, no entanto as Class podem ser usadas sem limites.
